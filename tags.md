## Status Tags

Used on missions, operations, projects, and quests to track lifecycle state.

| Tag           | Description                                                                                  |
| ------------- | -------------------------------------------------------------------------------------------- |
| `todo`        | Not yet started. Defined but no work has begun.                                              |
| `in-progress` | Actively being worked on.                                                                    |
| `on-hold`     | Paused intentionally. Will resume later — blocked, deprioritized, or waiting on something.   |
| `completed`   | Done. Delivered its intended outcome.                                                        |
| `dropped`     | Abandoned. No longer relevant or superseded by other work.                                   |
